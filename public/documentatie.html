Task Management API

Beschikbare Endpoints

  • POST /api/users: Voeg een nieuwe gebruiker toe.
  • GET /api/users: Haal alle gebruikers op (met paginering).
  • POST /api/tasks: Voeg een nieuwe taak toe.
  • GET /api/tasks: Haal alle taken op (met paginering).
  • GET /api/tasks/search: Zoek taken op titel of status.
